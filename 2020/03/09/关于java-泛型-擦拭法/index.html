泛型是一种类似”模板代码“的技术，不同语言的泛型实现方式不一定相同。

Java语言的泛型实现方式是擦拭法（Type Erasure）。

小结
Java的泛型是采用擦拭法实现的；

擦拭法决定了泛型：

不能是基本类型，例如：int；
不能获取带泛型类型的Class，例如：Pair.class；
不能判断带泛型类型的类型，例如：x instanceof Pair；
不能实例化T类型，例如：new T()。
泛型方法要防止重复定义方法，例如：public boolean equals(T obj)；

子类可以获取父类的泛型类型。

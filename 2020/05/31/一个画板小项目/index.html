一个画板小项目

2020-05-31 Java项目 Java 0 0 评论 字数统计: 4.3k(字) 阅读时长: 19(分)

这个画板项目是使用java编写的用于模拟Windows系统自带的画图工具的程序。包括如下的功能：设置画笔的粗细，选择背景颜色，绘制图形，清除图像，使用橡皮以及展开简笔画参考示例等功能。其中使用到的类以及技术包括：

  • Jframe窗体类
  • Canvas画板类的关键方法和事件
  • 窗体菜单栏的创建以及层级关系
  • 如何使用工具栏和按钮

    • 1、使用到的相关组件及技术

    其中用到的Swing组件有:
    JFrame窗体类，JWindows无边框窗体类，JButton按钮类，JToggleButton按钮类，JMenuBar菜单栏类，Jmenu菜单类，JMenuItem菜单项类

    Swing事件监听包括对鼠标点击事件监听以及鼠标拖拽事件监听。

    画图工具：

    Graphics2D绘图类

    其他技术：
    继承与封装
    IO
    Image文件类
    Ellipse2D圆形图形类
    Rectangle2D方形图形类

    DrawPictureFrame.java

    该类实现了画板项目的主要功能。

    以上的代码均详细的写了注释，便于理解和阅读

    DrawPictureCanvas.java

    PictureWindow.java

    该类主要是实现简笔画窗口功能。

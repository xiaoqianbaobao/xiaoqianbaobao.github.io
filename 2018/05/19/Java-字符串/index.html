在java中由双引号(“”)包围的都是字符串，不能作为其他数据类型使用，如”1+2”的输出结果不可能是3

10、判断字符串是否相等
对字符串进行比较不能简单地使用比较运算符”==”，因为比较运算符比较的是两个字符串的地址是否相同。即两个字符串的内容相同，两个对象的内存地址也是不同的，使用比较运算符仍然会返回false。
因此，比较字符串，应该用equals()方法和equalsIgnorCase()方法

14、正则表达式
正则表达式通常被用于判断语句中，用来检查某一字符串是否满足某一格式。
注意：在正则表达式中”.”代表任何一个字符，因此在正则表达式中如果想使用普通意义的点字符”.”，必须用转义字符”"

在正则表达式中可以使用方括号括起若干个字符来表示一个元字符，该元字符可代表方括号中的任何一个字符。例如，reg-“[abc]4”,这样的字符串a4,b4,c4都是和正则表达式匹配的字符串。方括号元字符还可以为其他格式。如:
[^456]:代表4、5、6之外的任何字符
[a-r]: 代表a~r中的任何一个字母。

例：在项目中创建类Judge，在主方法中实现使用正则表达式来判断指定的变量是否为合法的E-mail地址

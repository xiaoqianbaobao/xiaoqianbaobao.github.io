Win10 IIS配置PHP运行环境

Posted on 2018-08-10 | In 服务器脚本语言
如何在win10上利用IIS配置php运行环境

今天上午在w3school把asp学完了，感觉有些我还是不太明白，而且尽管我已经把IIS的asp运行环境搭建好了，有些asp代码还是无法正常运行，我也去Google了很多的方法，但还是无法解决我的难题。下午看来会php，突然觉得我为什么不使用php来作为服务器的脚本语言呢

PHP 是一门令人惊叹的流行语言！

它强大到足以成为在网络上最大的博客系统的核心（WordPress）！ 它深邃到足以运行最大的社交网络（facebook）！ 而它的易用程度足以成为初学者的首选服务器端语言！

可以看出php比asp更简单易学

PHP 能够做什么？

PHP 能够生成动态页面内容 PHP 能够创建、打开、读取、写入、删除以及关闭服务器上的文件 PHP 能够接收表单数据 PHP 能够发送并取回 cookies PHP 能够添加、删除、修改数据库中的数据 PHP 能够限制用户访问网站中的某些页面 PHP 能够对数据进行加密 通过 PHP，您可以不受限于只输出 HTML。您还能够输出图像、PDF 文件、甚至 Flash 影片。您也可以输出任何文本，比如 XHTML 和 XML。

可以看出php的作用不亚于asp

为什么使用 PHP？

PHP 运行于各种平台（Windows, Linux, Unix, Mac OS X 等等） PHP 兼容几乎所有服务器（Apache, IIS 等等） PHP 支持多种数据库 PHP 是免费的。请从官方 PHP 资源下载：www.php.net PHP 易于学习，并可高效地运行在服务器端

好了，废话不多说，我们既然要学习php，得要有一个php的运行环境吧，网上看了很多在win10搭建php运行环境的教程，总结了一下，我来写一个最适合新手小白的搭建教程。网上有基于apache,mysql的php搭建环境，但我觉得在win10来说，这个有点复杂了。既然win10有自己的IIS，我们直接利用IIS来创建php运行环境不就可以了吗。于是乎，我们来开始折腾吧！

安装IIS服务器

  1. 进入控制面板>>程序和功能>>打开或关闭Windows功能，找到Internet信息服务，加上CGI这一项.

下载安装配置PHP环境

  1. 下载PHP，官网地址:点击进入下载。
  2. 将下载的zip压缩包解压到F盘下，解压后为F:\php
  3. 进入控制面板>>管理工具>>Internet信息服务(IIS)管理器;
  4. 中间窗口选择"处理程序映射"双击，然后最右边选择"添加模块映射"

注意：在下载PHP时，要注意其说明选择合适的zip进行下载，注意下面这段英文，因为我们选择的是IIS，所以要选择Non-Thread safe (NTS)进行下载

Which version do I choose?

IIS
If you are using PHP as FastCGI with IIS you should use the Non-Thread Safe (NTS) versions of PHP.

配置文件

进入F:\php目录，冲命名文件php.ini-development改名为php.ini并复制到桌面

修改桌面的php.ini文件

打开php.ini文件后，使用快捷键ctrl+F搜索date.timezone,在后面将时区改为当前的时区，然后激活你需要激活的扩展选项，即将相应的语句前的分号去掉 再搜索extension_dir= 将其修改为extension_dir="F:\php\ext",修改并检查完毕后保存，然后把桌面的这个php.ini文件剪切到C:\Windows目录下即可.

и§Јеү–shadowsocksд№Ӣжәҗз ҒеҲҶжһҗ

2018-08-20 зҪ‘з»ң 0 0 иҜ„и®ә еӯ—ж•°з»ҹи®Ў: 2.4k(еӯ—) йҳ…иҜ»ж—¶й•ҝ: 9(еҲҶ)

еүҚйқўпјҢжҲ‘е·Із»ҸеңЁз§‘еӯҰдёҠзҪ‘дёҠеҜ№shadowsocksжңүдәҶдёҖе®ҡзҡ„зҗҶи§ЈпјҢе№¶еӨ§жҰӮжё…жҘҡжҳҜеҰӮдҪ•йҖҡиҝҮд»ЈзҗҶе®һзҺ°fqзҡ„пјҢдҪҶеҜ№дәҺshadowsocksжҳҜжҖҺж ·з”Ёpythonд»Јз Ғе®һзҺ°зҡ„пјҢжҲ‘иҝҳжҳҜдёҚжё…жҘҡпјҢеӣ жӯӨжҹҘйҳ…дәҶдёҚе°‘зҪ‘дёҠзҡ„ж–ҮжЎЈд»ҘеҸҠgithubдёҠйқўзҡ„жәҗз ҒпјҢеҜ№shadowsocksе®һиЎҢдәҶдёҖж¬Ўи§Јеү–пјҢжҸӯз§ҳshadowsocksзҡ„з§ҳеҜҶгҖӮ
йҖҡиҝҮжҲ‘зҡ„еҗ„з§ҚжҹҘйҳ…еҸ‘зҺ°пјҢзҺ°еңЁзҡ„shadowsocksе·Із»ҸжңүpythonпјҢc#,д»ҘеҸҠе®үеҚ“зӯүзҡ„дёҚеҗҢзүҲжң¬пјҢеҹәдәҺжҲ‘еҸӘеӯҰиҝҮpythonпјҢжүҖд»ҘйҖүжӢ©дәҶgithubдёҠpythonзүҲжң¬зҡ„иҝӣиЎҢеҲҶжһҗ
githubе®ҳж–№йЎөйқўshadowsocksжәҗз ҒдёӢиҪҪпјҢеҸ‘зҺ°пјҢshadowsocksе·ҘзЁӢ

ж ёеҝғд»Јз Ғ

е…¶е·ҘзЁӢж ёеҝғд»Јз ҒеқҮдҪҚдәҺ shadowsocks зӣ®еҪ•дёӢгҖӮе…¶е®ғж–Үд»¶е’Ңзӣ®еҪ•еҲҷжҸҗдҫӣдәҶжү“еҢ…гҖҒжөӢиҜ•зӯүеҠҹиғҪгҖӮ

еғҸ shadowsocks иҝҷз§Қд»Јз ҒйҮҸдёҚи¶ідёҖдёҮиЎҢзҡ„е°ҸеһӢе·ҘзЁӢпјҢйҖҡиҜ»жәҗз Ғд»…йңҖеҮ дёӘе°Ҹж—¶пјҢйҳ…и§ҲйЎәеәҸд№ҹжІЎжңүзү№еҲ«зҡ„и®Із©¶гҖӮдҪҶжҳҜпјҢеҪ“йҒҮеҲ°д»Јз ҒйҮҸеҚҒдёҮиЎҢз”ҡиҮізҷҫдёҮиЎҢзҡ„еӨ§еһӢе·ҘзЁӢж—¶пјҢйҳ…иҜ»е…ЁйғЁд»Јз ҒжҳҜдёҖд»¶дёҚеҸҜиғҪе®ҢжҲҗзҡ„д»»еҠЎпјҢз”ЁжӯЈзЎ®зҡ„йЎәеәҸжөҸи§Ҳе°ұеҸҳеҫ—е°ӨдёәйҮҚиҰҒгҖӮдёҖиҲ¬жқҘиҜҙпјҢжү“ејҖдёҖдёӘйҷҢз”ҹзҡ„е·ҘзЁӢпјҢжңҖеҘҪе…Ҳе®ҡдҪҚе…¶ main еҮҪж•°гҖӮжҠ“дҪҸдәҶзЁӢеәҸзҡ„е…ҘеҸЈпјҢе°ұжҠ“дҪҸдәҶйҖ»иҫ‘зҡ„ж №иҠӮзӮ№пјҢжӯӨеҗҺеҶҚеҲҶеҲ«иҝҗз”Ёе№ҝеәҰдјҳе…ҲжҗңзҙўгҖҒж·ұеәҰдјҳе…ҲжҗңзҙўгҖҒеӣһжәҜжҗңзҙўзӯүеӨҡз§ҚжүӢж®өпјҢйҖҗжӯҘжҸҸз»ҳеҮәе·ҘзЁӢзҡ„еӨ§дҪ“иҪ®е»“гҖӮ

shadowsocks еңЁ local.py е’Ң server.py дёӨеӨ„еҲҶеҲ«е®ҡд№үдәҶ main еҮҪж•°гҖӮеӣ дёәиҝҷдёӨдёӘ main еҮҪж•°еұһдәҺдёҚеҗҢзҡ„жЁЎеқ—пјҢиҝҷжҳҜе®Ңе…ЁеҗҲжі•зҡ„гҖӮеңЁ shell.print_help дёӯпјҢе®ғд»¬иў«еҲҶеҲ«иөӢдәҲдәҶ sslocal е’Ң ssserver иҝҷдёӨдёӘеҗҚеӯ—пјҢжҲ‘д»¬д№ҹеҸҜд»Ҙз§°д№Ӣдёәе®ўжҲ·з«Ҝе’ҢжңҚеҠЎз«ҜгҖӮеҜ№жҜ”жҹҘзңӢ local.py е’Ң server.pyпјҢжңҚеҠЎз«ҜдёәдәҶж”ҜжҢҒеӨҡз»„з«ҜеҸЈе’ҢеӨҡиҝӣзЁӢпјҢеҶҷдәҶдёҚе°‘йўқеӨ–зҡ„д»Јз ҒгҖӮеҰӮжһңжҠҠиҝҷдёӨйғЁеҲҶеүҘзҰ»жҺүпјҢеү©дёӢзҡ„дё»е№Із»“жһ„дёҺе®ўжҲ·з«Ҝе·®ејӮз”ҡеҫ®пјҡ

еңЁ main еҮҪж•°дёӯпјҢshadowsocks и§Јжһҗй…ҚзҪ®ж–Үд»¶жҲ–е‘Ҫд»ӨиЎҢеҸӮж•°еҗҺпјҢйҰ–е…ҲжіЁеҶҢдәҶзі»з»ҹе®ҲжҠӨиҝӣзЁӢпјҢд»ҘдҫҝдәҺеңЁеҗҺеҸ°иҝҗиЎҢгҖӮд№ӢеҗҺпјҢе®ғеҲӣе»әдәҶ DNS и§ЈжһҗеҷЁгҖҒTCP дёӯз»§еҷЁгҖҒUDP дёӯз»§еҷЁдёүдёӘз»„д»¶пјҢе№¶з”ЁдәӢд»¶еҫӘзҺҜе°ҶиҝҷдёүиҖ…з»ҹдёҖиө·жқҘгҖӮдәӢд»¶еҫӘзҺҜејҖе§ӢеҗҺпјҢиҝҷдәӣз»„д»¶дјҡеңЁеӨ–йғЁдәӢд»¶зҡ„и§ҰеҸ‘дёӢе®һзҺ°йў„е®ҡзҡ„иЎҢдёәпјҢзӣҙиҮізЁӢеәҸйҒӯйҒҮеҶ…йғЁй”ҷиҜҜжҲ–еӣ жҚ•иҺ·еӨ–йғЁдҝЎеҸ·иҖҢйҖҖеҮәгҖӮ

е®ўжҲ·з«ҜдёҺжңҚеҠЎз«Ҝзҡ„еҢәеҲ«еңЁдәҺдј з»ҷ TCPRelay е’Ң UDPRelay зҡ„з¬¬дёүдёӘеҸӮж•° is_local зҡ„еҖјдёҚеҗҢгҖӮд№ҹе°ұжҳҜиҜҙпјҢе®ғд»¬зҡ„иЎҢдёәе·®ејӮиҰҒиҝӣе…ҘдёҠиҝ°дёӨдёӘзұ»дёӯжүҚиғҪзңӢеҮәжқҘгҖӮд»Һд»Јз ҒдҪ“йҮҸеҮәеҸ‘пјҢе…ҲеҲҶжһҗ UDPRelay жҳҫз„¶жҳҜдёӘжӣҙжҳҺжҷәзҡ„йҖүжӢ©гҖӮеңЁжһ„йҖ еҮҪж•°йҮҢпјҢжҲ‘д»¬зңӢеҲ°дәҶиҝҷж ·зҡ„д»Јз Ғпјҡ

дёҚйҡҫзңӢеҮәпјҢshadowsocks зҡ„ж•ҙдҪ“з»“жһ„еҸҜд»Ҙз”ЁдёӢеӣҫиЎЁзӨәеҮәжқҘ

е®ўжҲ·з«Ҝ sslocal зӣ‘еҗ¬ local_address:local_port з«ҜеҸЈгҖӮеҪ“з”ЁжҲ·иҜ·жұӮдәӢд»¶и§ҰеҸ‘ж—¶пјҢе®ўжҲ·з«ҜиҜ•еӣҫд»ҺиҜҘз«ҜеҸЈиҜ»еҸ–ж•°жҚ®пјҢеҠ еҜҶеҗҺеҸ‘йҖҒиҮі server:server_portгҖӮжңҚеҠЎз«Ҝ ssserver ж”¶еҲ°е®ўжҲ·з«Ҝзҡ„иҜ·жұӮеҗҺпјҢи§ЈеҜҶж•°жҚ®еҶ…е®№пјҢи§Јжһҗзӣ®ж Ү IP ең°еқҖпјҢжҠҠз”ЁжҲ·зҡ„иҜ·жұӮиҪ¬еҸ‘иҮізӣ®ж ҮжңҚеҠЎеҷЁгҖӮеҪ“ ssserver ж”¶еҲ°зӣ®ж ҮжңҚеҠЎеҷЁзҡ„еӣһеә”еҗҺпјҢеҶҚжҠҠиҝҷдәӣдҝЎжҒҜеҠ еҜҶе№¶йҖҒеӣһе®ўжҲ·з«ҜпјҢз”ұе®ўжҲ·з«ҜжңҖз»Ҳе“Қеә”з”ЁжҲ·зҡ„иҜ·жұӮгҖӮ

еҖјеҫ—жіЁж„Ҹзҡ„жҳҜпјҢиҷҪз„¶е®ўжҲ·з«Ҝе’ҢжңҚеҠЎз«ҜйғҪжңү DNS и§ЈжһҗеҷЁпјҢдҪҶд»–д»¬жүҝжӢ…зҡ„иҙЈд»»жңүеҫҲеӨ§зҡ„е·®ејӮгҖӮе®ўжҲ·з«Ҝзҡ„ DNS з”ЁдәҺи§ЈжһҗжңҚеҠЎз«Ҝзҡ„ IP ең°еқҖпјҢеҰӮжһңеңЁй…ҚзҪ®ж–Үд»¶дёӯдҪҝз”Ё IP д»ЈжӣҝеҹҹеҗҚпјҢеҲҷе®ўжҲ·з«Ҝзҡ„и§ЈжһҗеҷЁжҳҜдёҖдёӘеҸҜд»Ҙз§»йҷӨзҡ„з»„д»¶гҖӮжңҚеҠЎз«Ҝзҡ„ DNS е°Ҷз”ЁжҲ·иҜ·жұӮдёӯзҡ„еҹҹеҗҚзҝ»иҜ‘жҲҗзӣ®ж ҮжңҚеҠЎеҷЁзҡ„ IP ең°еқҖпјҢжҳҜеҝ…иҰҒзҡ„з»„д»¶гҖӮиҖғиҷ‘еҲ°жңҚеҠЎз«Ҝзҡ„йғЁзҪІең°зӮ№пјҢйҖҡеёёдёҚдјҡеҸ—еҲ° DNS жұЎжҹ“зҡ„еҪұе“ҚпјҢеӣ жӯӨеҸҜд»Ҙе®үе…Ёең°дҪҝз”Ёзі»з»ҹй»ҳи®Өзҡ„ DNS жңҚеҠЎеҷЁгҖӮ

读《呼兰河传》笔记

2018-04-26 文学 文学赏析读后感 0 0 评论 字数统计: 1k(字) 阅读时长: 3(分)

萧红的书，这是我第一次看，觉得相比较以前的女作家，没有一人能及她。那些文字，是再平常不过的了，却能点点滴滴刻画出每个情感，每个性格，在心里一点一点蚀出一个大坑，空落落的只想落泪

夏天在图书馆读《呼兰河传》是一种怎样的感受，没开空调却觉得寒气逼人，全身并未感觉到一丝暑气，你能透过文字穿越到那个时代，去感受北方土地上农民的炎凉冷暖

常言道人生苦乐参半，有时乐观一点，就把吃苦当吃补，斩钉截铁对自己说只要努力奋斗总会有柳暗花明的一天。憋着忍着，不敢松懈，不敢退缩，脸上带笑，心也要拿钢筋水泥加固过，就怕一旦动摇再难鼓起勇气。这人生啊，说不艰辛是骗人的。若碰上某个深夜彷徨的时候，看到了凄婉的文字，不自觉就会放缓了呼吸，放松了情绪，任寂寞席卷而来。有时晚上寂寞难耐，总会对自己说一句，反思过往，是不是书又读少了，日子过的不充实了

“请神的人家为了治病，可不知那家的病人好了没有？却使邻居街坊感慨兴叹，终夜而不能已的也常有。
漫天星光，满屋月亮，人生如何，如此悲凉”

“若赶上一个下雨的夜，就特别凄凉，寡妇可以落泪，渔夫就要起来彷徨。
那鼓声就好像故意招惹那般不幸的人，打的有急有慢，好像一个迷路的人在夜里诉说着他的迷茫，又好像不幸的老人在汇着他幸福的短短的幼年。又好像慈爱的母亲送着她的儿子远行。又好像是生离死别，万分的难舍
人生为了什么，才有这样凄凉的夜。”

“逆来顺受，你说我的生命可惜，我自己却不在乎。你看着很危险，我却自己以为得意。不得意怎么样？人生是苦多乐少。”

东大街的泥坑，陷了多少次马。也没有人说要去填；漏粉人家的草房歪的一塌糊涂，也没见有人说要去修；七月十五呼兰河上的水灯慢慢向下游漂，漂着漂着就灭了一盏；邻家人的歌声，像一朵花开在墙头上，越是鲜明，就越觉得凄凉，竟是生也由不得人，死也由不得人。

满篇满纸，都是回忆，慢眼满心，都是萧索。芸芸众生，各有自己的无奈挣扎，保不准什么时候一个念头就放任自己坠落。跌进无边黑暗里，永远永远都不想醒来。而我绝对不能这样，我连那个崖边都不能靠近。一切脆弱，都只在今夜。

但，蚀出的大坑，可以填上；松动的围墙，可以修补；悲凉的人生，却又什么办法补救

想必她是经历了怎样的人生才让自己的所书写的每一个文字都充满了悲凉和，《呼兰河传》真的是东北那片黑土地在那片封闭的角落里一张悲怆的回忆，唯一疼爱萧红的怕也就只有爷爷了。

想到看那团圆家媳妇的时候，看到那些人的愚昧无知，思想的落后，才导致这样一个本来青春洋溢的女子，被糟蹋的不成样子，我想不仅仅是呼兰河吧，在那个时代，中国大多还是处在半封建时代，人们的思想仍然还没有得到解放，人民还是过着水深火热的生活，女性的权利没有得到解放
